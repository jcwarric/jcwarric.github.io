Processing

  • Flexible software sketchbook for learning to code within the context of the visual arts
  • Created by Ben Fry and Casey Reas in 2001
  • Language based on Java, IDE
  • Promotes software literacy in the visual arts
  • Promotes visual literacy within technology
  • Used by thousands of students, artists, designers, and hobbyists
  • Free and open source
  • For Linux, Mac OS X, Windows, Android, and ARM
  • Over 100 libraries
  • Well documented
  • Google, Intel have used Processing for prototyping new interfaces and services
