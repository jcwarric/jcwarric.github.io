My Favorite HTML tags

Introduction

The purpose of this website is to discuss my top three favorite html tags and demonstrate their usage.

<Canvas>

Your browser does not support canvas

I really like the <Canvas> tag because of its ability to draw dynamic content to the screen. The < Canvas > tag allows the web developer to be creative and artistic when making animations. The animation above was inspired by Casey Reas' Network A . I had a lot of fun researching how the canvas works and playing around with making a simplified version of Reas' animation work on my own website.

The < SVG >Tag

I really like the svg tag because it allows the developer to dynamically resize vector-based images. Unlike raster images, svg images don't become pixelated when their size is increased. Vector-based graphics programs like Adobe Illustrator and Inkscape make creating complex svg images easy and fun.

The <div>Tag

While it may seem kind of boring, I really like <div> because it is useful for grouping diverse html objects so that they can be modified by a CSS or javascript file. I used divs several times on this page, such as grouping the svg image and paragraph text above.
