Explanations

Navbar

I used the same navbar on each page. I formatted it with several other bootstrap classes including navbar-fixed-top to keep the navigation bar at the top of the page even when the user scrolls, dropdown-menu and dropdown-item to format the PAT navigation link so that it has sub-links, navbar-collapse to make the navigation bar collapse when viewed with a small screen, among others.

Carousel

I used the carousel class on the home page to create an image gallery header for the header. I really like the aesthetic of having multiple transitioned images because it's more interesting than a static image and can be used to highlight the many aspects of the community that my website is meant to describe (mass celebrations, service and mission events, etc). I used the carousel slide, carousel-inner, and item classes to create the individual elements of the carousel. The img-responsiveclass was also useful in making sure that each image fit the carousel slides

Cards

I used cards on the homepage to display each of the four parishes that are part of the Midland Area Catholic Community group. To format the cards, I used the card, card-body, and card-title classes.

Collapse

I used an accordion with cards on my Meeting Minutes page to allow the user to toggle a particular meetings' notes by date. It currently looks underwhelming, but I think that the aesthetic of using a collapsable set of cards would improve with more than one entry. I chose to use an accordion because the meeting notes are about a page long each. Since the site is meant to act as a repository for all future meeting notes, displaying the full entries would make the page too long cumbersome to scroll through. Some classes that I used to format the accordion are: accordion,card, collapse, card-body, and btn-link.

Forms

I used a form on the Contact page to allow the site users to submit a message (in theory - I didn't add any code to handle the actual data entry actions). Classes that I used to implement this form include: form-group, form-check-input, form-check-label, and form-control.

Discussion

Bootstrap and WordPress are both great tools for web development. Both have great documentation and are popular on user forums like stack-overflow, which are great for troubleshooting.

Personally, I like Bootstrap a lot more than WordPress because I haven’t put much time into learning about how to use WordPress themes effectively. While working on homework 1, I often struggled to get items to display on my pages because I hadn’t understood that WordPress themes enforce their structures. While I always sketch out a design of my website before coding it, I usually change elements while writing the code. This was difficult for me to do in WordPress because I was tied-in to the theme that I selected at the beginning. As a programmer, I also feel a lot prouder of a website when I have coded it myself, rather than following a pre-coded template.

However, overall WordPress would be a better tool for my website because I don’t want to be involved in maintaining it/updating content long-term and instead would be handing it to the MACC team to manage, none of whom know how to code. The user-friendly nature of WordPress would allow a member of the team to easily update content on his/her own. Using a WordPress theme is also beneficial because I want my website to look professional. WordPress has a wide variety of professional-looking themes to choose from that are completely responsive without making me set breakpoints using media-queries or bootstrap’s grid classes.

WordPress also has great widgets. I really liked the Google Calendar on my WordPress site better than my Bootstrap site because I used a widget for it. Not only did it display more nicely, it also automatically synched with an “upcoming events” section to display events from the associated calendar, an effect that I could only do manually (or with a lot of effort) in Bootstrap.

I recently found an article on creating a WordPress theme using Bootstrap that I wanted to try out. While I’m still trying to learn how to create aesthetically appealing web designs, the ability to make a theme in bootstrap and then give it to a client in a WordPress-friendly format would be highly useful and practical.
